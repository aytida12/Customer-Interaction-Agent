An AI-powered SMS chatbot that automates customer interactions, qualifies leads, checks availability, books appointments on Google Calendar, and saves leads to Google Sheets. **Production-ready, copy-paste deployable.**

**Architecture:**
- **Twilio** for SMS ingestion
- **OpenAI GPT-4o-mini** for natural language understanding and function calling

- **Google Sheets** as CRM backend
- **Express.js** server with webhook handler
- **Conversation history** for context retention
